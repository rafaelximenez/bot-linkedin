# Bot Linkedin

[PT] Bot para operar no linkedin.<br>
[EN] Bot to operate on linkedin.


## Funcionalidades:
* Realizar login
* Pesquisar vagas

## Como usar:

### Instalação
```
    pip install -r requirements.txt
```

### Configuração - config.py
```
    URL = 'http://www.linkedin.com/login'
    EMAIL = ''
    PASSWORD = ''
```
